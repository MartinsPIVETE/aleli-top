Sim, o Bitbucket oferece suporte ao português em sua interface de usuário e documentação, sendo uma plataforma de hospedagem de código desenvolvida pela Atlassian. 
Aqui estão os detalhes sobre o suporte ao português no Bitbucket:
Interface em Português: O site e o ambiente de trabalho do Bitbucket podem ser configurados para o português, facilitando a navegação para usuários brasileiros.
Documentação e Suporte: A Atlassian disponibiliza guias, tutoriais e o centro de suporte com bastante conteúdo traduzido para o português.
Integração com ecossistema: Como parte das ferramentas da Atlassian (assim como o Jira e Trello), o Bitbucket é amplamente utilizado no Brasil e, portanto, adaptado ao idioma. 
Embora a interface possa ser configurada para português, a maioria dos comandos técnicos do Git e os nomes de funcionalidades técnicas (como Pull Request, Branch, Pipelines) continuam em inglês, o que é padrão na indústria de desenvolvimento. 

"Issues" em português traduz-se comumente como questões, assuntos ou problemas, dependendo do contexto. É um termo versátil usado para tópicos de discussão (assuntos), complicações (problemas), edições de publicações ou emissões de documentos. É uma palavra mais formal e diplomática que problem. 
Principais significados e traduções:
Questões/Assuntos: Frequentemente usado em debates, política ou reuniões de negócios, ex: "The latest issues" (As últimas questões).
Problemas: Refere-se a dificuldades, complicações ou questões pessoais/emocionais, ex: "Personal issues" (Problemas pessoais).
Edições/Publicações: Refere-se a números de revistas ou jornais, ex: "The June issue" (A edição de junho).
Emissão: Relacionado à ação de expedir documentos, ex: "Date of issue" (Data de emissão). 
Exemplos de uso:
"Environmental issues" (Questões ambientais).
"We have some issues with the project" (Temos alguns problemas/questões com o projeto).
"The company has many issues" (A empresa tem muitos assuntos/problemas). 

"Pull" em português significa principalmente puxar. Como verbo, também pode traduzir-se por arrastar, tirar, sacar, atrair, distender ou, no sentido figurado, influenciar. Como substantivo, pode significar puxada, puxão, tração ou atração. 
Principais usos e traduções de "pull":
Puxar/Mover para si: Pull the door (Puxe a porta).
Puxão (Substantivo): A strong pull (Uma puxada forte).
Influência/Atração: Gravitational pull (Atração gravitacional).
Sacar (arma/objeto): Pull a gun (Sacar uma arma).
Arrancar/Tirar: Pull off (Arrancar/Tirar). 
Expressões Comuns:
Pull someone's leg: Brincar/provocar alguém.
Pull strings: Mexer os pauzinhos (usar influência).
Pull the plug: Cancelar/acabar com algo. 
Nota: PULL é o oposto de PUSH (empurrar).
